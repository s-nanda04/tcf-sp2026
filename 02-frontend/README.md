# Lesson 2: Frontend Crash Course

**Date:** XX Feb 2026

## Overview
The "Frontend" is everything a user sees and interacts with. We will move from basic documents (HTML) to styled interfaces (CSS) to interactive applications (React).

## Agenda
1. **The Trinity of Web**
   - **HTML**: The Skeleton (Nouns). `<div>`, `<h1>`, `<button>`.
   - **CSS**: The Skin (Adjectives). `color: red;`, `display: flex;`.
   - **JavaScript**: The Brains (Verbs). `onClick`, `fetch()`.

2. **Intro to React**
   - Why React? (Component reusability, Virtual DOM).
   - **JSX**: Writing HTML inside JavaScript.
   - **Components**: Building blocks (Header, Footer, Button).
   - **Props**: Passing data down (Parent -> Child).
   - **State**: Remembering data (e.g., a counter).

## Class Activity
We will build a simple "Counter" button together to understand State.
